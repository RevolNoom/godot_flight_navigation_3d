# GODOT MORTON CODE
Provide Morton code implementations in 2D and 3D.

## Features
- Encoding/Decoding implemented with Magic bits.
- x, y, z setters.
- Add/Subtract x/y/z.
- Increment/Decrement x/y/z.
- Comparisons: Less, less-equal, greater, greater-equal.
